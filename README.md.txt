# Digital Curator

A beautiful and modern digital asset management web application built with Tailwind CSS.

![Digital Curator Preview](https://via.placeholder.com/800x400/4f46e5/ffffff?text=Digital+Curator+Preview)

## ✨ Features

- **Clean & Modern UI** — Elegant design with soft shadows, rounded corners, and smooth interactions
- **Storage Dashboard** — Real-time storage usage with beautiful progress visualization
- **File Breakdown** — Visual categorization of Photos, Documents, and Others
- **Collections Management** — Browse your media in organized folders (Images, Videos, Documents)
- **Recently Curated** — Quick access to your latest files with hover animations
- **Responsive Design** — Fully optimized for desktop, tablet, and mobile
- **Premium Upgrade Flow** — Attractive call-to-action for upgrading storage
- **Material Symbols** — Beautiful Google Material Icons with proper fill variations
